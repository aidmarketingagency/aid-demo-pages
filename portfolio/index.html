Conversational AI Lead Qualification Appointment Booking SMS / Voice

AI Receptionists — Missed-Call Takeover

When a contractor misses a call, an AI receptionist immediately texts back, qualifies the lead with targeted questions, and books an appointment — all via SMS conversation, no human required. The system handles objections, captures job scope, and writes the booked appointment directly to the contractor's calendar. Architecture: missed call triggers a PayMeGPT voice/SMS workflow; Claude drives the conversational qualification via a webhook-connected Netlify Function; confirmed bookings sync to the contractor's Google Calendar via GCal API with a confirmation SMS to both parties.

Infrastructure Multi-Agent Memory System Scheduling

Agent Memory + Scheduling Infrastructure

The operational backbone under all the other builds: a 10-agent topology with cross-session memory, two-lane scheduling (GitHub Actions for deterministic recurring jobs; Air task queue for judgment work), and a Supabase semantic core that consolidates facts across agent sessions. Agents share state, avoid redundant work, and recover from interruption without losing context. Python stop hooks write session facts to Supabase on every agent exit; a nightly GitHub Actions workflow consolidates and pins the semantic core; model routing is enforced by a hook layer that gates which model each task class may use.
